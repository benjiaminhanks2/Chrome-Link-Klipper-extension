FAQ

What is Link Klipper ?

Link Klipper is a chrome extension to extract unique links on a specific web page and store it in a csv or a txt format.

What are the options on the settings page ?

The format in which one wants to download the file, if links should match a specific regular expression and only those links should be exported and if you want specific file formats extension links to be exported can be managed via the Settings page.

Link Klipper is a simple yet very powerful chrome extension which helps you extract all the unique links on a webpage and export them to a file. No more hassle to copy each link from a webpage and store it individually. This plugin will add a right click context menu to your browser. By right clicking on any part of the webpage, you will be able to access the options under the context menu entry "Link Klipper - Extract Links".
Applications :
  • For people who work in the SEO industry, you can quickly collect a group of links on the page
  • While browsing through images on any page, you can download the target links of all the images
  • While researching online, you can gather useful resourceful links and store them for future reference to a CSV file
